音乐拼图游戏快捷键指南
为了提升您的游戏体验，音乐拼图游戏支持以下快捷键操作，方便您更快速地进行游戏。

全局快捷键
空格键（Space）：停止所有音频播放
功能：当您正在播放音乐片段或拼图时，按下空格键即可立即停止所有正在播放的音频。
使用场景：需要快速停止音频时，按下空格键，无需移动鼠标或切换焦点。
导航快捷键

Tab 键：移动焦点到下一个可交互元素
功能：将当前焦点移动到下一个可交互的界面元素，如按钮、链接或输入框。
使用场景：快速遍历界面上的所有可交互元素，逐一进行选择或操作。

Shift + Tab 键：移动焦点到上一个可交互元素

功能：将当前焦点移动到上一个可交互的界面元素。
使用场景：返回到前面的元素，重新选择或修改之前的操作。

向右箭头键（→）：焦点移动到下一个元素

功能：在特定的区域内（如音乐片段列表或拼图区域），将焦点移动到下一个元素。
使用场景：在同一组元素中（如按钮或拼图片段）快速导航。

向左箭头键（←）：焦点移动到上一个元素

功能：在特定的区域内，将焦点移动到上一个元素。
使用场景：返回查看前面的元素，或者重新选择之前的按钮或片段。
操作快捷键

Enter 键：激活当前焦点元素

功能：当焦点位于按钮或拼图片段上时，按下 Enter 键可激活该元素的主要功能，例如播放音频或添加拼图片段。
使用场景：无需使用鼠标点击，直接通过键盘操作当前选中的元素。
Delete 键 或 Backspace 键：删除选中的拼图片段

功能：当焦点位于拼图区域的某个拼图片段上时，按下 Delete 或 Backspace 键可将其从拼图中移除。
使用场景：需要调整拼图内容，快速删除不需要的片段。
功能快捷键

Alt + P：播放拼图

功能：依次播放拼图区域中已添加的所有音乐片段。
使用场景：在拼图完成或调整后，快速检验拼图的整体效果。

Alt + R：重置拼图

功能：清空拼图区域，移除所有已添加的拼图片段。
使用场景：需要重新开始拼图，或者放弃当前拼图时。
片段选择快捷键

Alt + 数字键（1 到 9）：选择并添加对应的音乐片段

功能：快速选择指定的音乐片段，播放并将其添加到拼图区域。
使用场景：无需使用鼠标点击，直接通过键盘添加音乐片段。
示例：

Alt + 1：选择第一个音乐片段。
Alt + 2：选择第二个音乐片段。
...
Alt + 9：选择第九个音乐片段。
快捷键使用提示


焦点管理：

Tab 键和 Shift + Tab 键可用于在整个页面的可交互元素之间循环移动焦点。
**方向箭头键（← / →）**主要用于在同一组元素中（如音乐片段按钮或拼图片段）移动焦点。
激活元素：

当焦点位于按钮或拼图片段上时，按下 Enter 键 或 空格键 可激活该元素的主要功能。
避免输入冲突：

在使用快捷键时，请确保当前未在文本输入框或数字输入框中，以免影响输入或导致快捷键无法生效。
组合键的使用：

按下组合键时，请先按住 Alt 键，然后再按相应的字母键或数字键。
常见操作速览
停止所有音频：按下 空格键。

导航焦点：

下一个元素：按 Tab 键 或 →（右箭头键）。
上一个元素：按 Shift + Tab 键 或 ←（左箭头键）。
激活元素：按 Enter 键 或 空格键。

删除拼图片段：在拼图片段上，按下 Delete 或 Backspace 键。

播放拼图：按下 Alt + P。

重置拼图：按下 Alt + R。

添加音乐片段：按下 Alt + 数字键（1-9），选择并添加对应的音乐片段。

额外提示
添加拼图片段的顺序：由于音乐片段的标签与原始顺序无关，建议您仔细聆听每个片段，以判断其在原始音乐中的位置。

拼图完成验证：在添加完所有拼图片段后，按 Alt + P 播放拼图，系统会自动检查拼图顺序是否正确，并给予反馈。

语音提示：游戏内置了语音提示功能，在执行某些操作时会提供音频反馈，帮助您了解当前的游戏状态。
